# Raybot

Telegram bot on Raydium

## How to use

1. Get your bot token from BotFather
2. Create a `.env` file in the root directory and add the following line:
```
BOT_TOKEN=<your_bot_token>
```
3. Run the bot with docker
```bash
docker compose up -d
```
